# Yesodos Static Website

Clean, responsive static website for Yesodos, built with plain HTML, CSS, and JavaScript.

## Files

- `index.html`
- `about.html`
- `services.html`
- `contact.html`
- `style.css`
- `script.js`
- `assets/yesodos-logo.jpg`
- `assets/flyer-student.jpg`

## Preview Locally

Open `index.html` in a browser. No build step is required.

## Deploy With GitHub and Netlify

1. Create a new GitHub repository.
2. Upload all files and the `assets` folder to the repository.
3. In Netlify, choose **Add new site** and then **Import an existing project**.
4. Connect the GitHub repository.
5. Leave the build command blank.
6. Set the publish directory to the repository root.
7. Deploy the site.

For future edits, push changes to GitHub and Netlify will redeploy automatically.
